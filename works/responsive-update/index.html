Responsive Update

Type レスポンシブ対応
Created Date 2026.01
企画・デザイン 1週間
実装・修正 1.5週間
使用技術 HTML / CSS

制作概要 Overview

PC表示にのみ最適化されていた既存のコーポレートサイトを、スマートフォンやタブレットでも快適に閲覧・操作できるよう、フルレスポンシブ対応へと改修しました。既存のデザインの良さを活かしつつ、モバイルファーストの視点でUIを再構築しました。

サービス設定 Premise

・モバイル端末からのアクセス増加に伴うユーザビリティの向上
・SEO対策を考慮した、管理しやすく堅牢なコードへのリプレイス

ターゲット Target

移動中や隙間時間にスマートフォンで情報収集を行う、企業の既存顧客および新規取引先担当者

目的 Purpose

デバイスを問わず一貫した情報提供を行い、サイトの回遊率向上と問い合わせ数の増加を図る

コンセプト Concept

・テーマ：どこでも、変わらない快適さを
・デバイスごとに最適化された読みやすいレイアウト
・BEMを用いたメンテナンス性の高いCSS設計

サイト構成 Structure

全デバイスで共通のHTMLを使用し、CSSのメディアクエリでスタイルを切り替える手法を採用しました。特にメニュー周りはスマホ時にハンバーガーメニューへ切り替わるよう設計し、限られた画面幅でも主要なコンテンツへ素早くアクセスできるよう構造化しています。

デザイン Design

小さな画面でも視認性が損なわれないよう、フォントサイズや行間をミリ単位で調整しました。こだわりである「枠線や下線の削除」を徹底し、セクションの区切りは背景色の微細な変化や余白のコントロールだけで表現しています。これにより、スマートで現代的なWebサイトへと生まれ変わりました。
